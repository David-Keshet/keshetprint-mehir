# קשת פרינט - מחירון

אתר חישוב מחירים להדפסת חוברות וחומרים נלווים.

## קבצים עיקריים

- `Booklet.html` - מחשבון חוברות (עמוד ראשי)
- `CAL.html` - מחשבון כללי
- `code.html` - דף קוד
- `מחשבון חוברות.html` - גרסה נוספת של מחשבון החוברות

## התקנה והפעלה

אתר זה פועל כאתר סטטי ואינו דורש התקנה. ניתן לפתוח את קבצי ה-HTML ישירות בדפדפן.

## פריסה ב-Vercel

האתר מוגדר לפריסה אוטומטית ב-Vercel עם:
- תמיכה בעברית (RTL)
- עיצוב רספונסיבי
- ניתוב אוטומטי לעמוד הראשי
